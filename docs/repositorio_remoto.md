# Creación y vinculación de un repositorio remoto en GitHub

        Accedemos al sitio web de GitHub y procedemos a registrar una cuenta nueva (si no la tenemos).

    En la parte superior derecha, hacemos clic en el ícono “+” y seleccionamos la opción “New repository”.

    Completamos los campos obligatorios, como el nombre del repositorio, su descripción y la visibilidad (público o privado).

    Una vez llenado el formulario, pulsamos “Create repository” para generarlo.

    GitHub nos mostrará un conjunto de comandos para enlazar nuestro repositorio local con el recién creado.

    En la terminal se ejecuta (git remote add origin (enlace proporcionado por GitHub))

    para finalizar, se sube el contenido con el comando (git push -u origin main)
