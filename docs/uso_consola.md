# descripcion de conceptos aprendiods sobre como usar la consola para navegar y crear directorios y archivos.

    Para la creacion y navegacion de archivos utilizamos los comandos de pwd que muestra el directorio actual, ls que es para lista de archivos y carpetas dentro de un directorio, cd para cambiar de directorio, cd .. para regresar al directorio anterior, mkdir(nombre) este crea un nuevo directorio.

    touch(archivo) crea un nuevo archivo, cat(archivo) muestra el contenido de un archivo.
